Intended for Tour Operators, the MyTourOp Client connects with MyTourOp.com to manage enquiries from tourists.

== Description ==

With MyTourOp, tourists can make enquiries on your Wordpress website. Enquiries will be stored in the MyTourOp backend system. From there, you can manage payments, testimonials and statistics linked to this enquiry.


**Features**

* Make any tour itinerary "bookable" with the [bookable] shortcode
* Process credit card payments using the PayPal gateway
* Refine your advertisement campaigns using the detailed enquirer statistics
* Manage testimonials


== Installation ==

1. Download, unzip and upload to your WordPress plugins directory
2. Activate the plugin within you WordPress Administration
